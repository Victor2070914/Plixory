## ⚡ Core Architecture

The Plixory platform operates as a unified intelligence layer split into two synchronized core subsystems:

### 1. Plixory Search AI
A powerful knowledge engine built to structure web data into comprehensive, real-time verified responses.
* **Multi-Source Retrieval:** Seamless background data ingestion.
* **Structured Reasoning:** Breaks down complex queries into logical steps.
* **Cited Answers:** High-integrity outputs with built-in reference structures.

### 2. Plixory Emotional AI
A deep conversational engine designed to understand user intent, tone, and emotional undertones.
* **Emotion Perception:** Adapts conversation styles dynamically based on text sentiment.
* **Contextual Memory:** Tracks overarching themes across interactions.
* **Human-Centric Dialogue:** Built to prioritize natural conversational flow over standard rigid prompt-response frameworks.

## 🛠️ Tech Stack & Architecture

* **Frontend Engine:** Semantic HTML5 & Modern CSS Layering
* **Styling Framework:** Tailwind CSS (via CDN configuration)
* **Design Aesthetic:** High-fashion "Quiet Luxury" & Neo-brutalist tech elements with dark ambient glow gradients
* **Database Layer:** Asynchronous JavaScript Fetch API linked to an external REST API data broker (SheetDB engine powering Google Sheets integration)
